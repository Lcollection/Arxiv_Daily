    早期生命血压与中晚年认知功能：一项合成纵向队列分析 Bustillo, A. J. 2026-07-06 PDF 背景：高血压是已被充分研究的痴呆风险因素。高血压对认知老化的影响可能因生命阶段而异，但既往研究多聚焦于中年或晚年血压。这一空白源于缺乏同时涵盖儿童期血压测量与晚年认知评估的纵向队列。为填补该空白，我们采用合成队列方法，探究早期血压与中晚年认知的关联。方法：以数据融合为指导框架，我们将提供早期血压数据（4-16岁）的博加卢萨心脏研究（BHS）与提供中晚年认知数据（58-70岁）的CARDIA研究合并，构建合成纵向队列。两队列在17-57岁年龄段存在重叠。认知评估包括执行功能、记忆力和处理速度的原始分与Z分数。早期收缩压（SBP）定义为4-16岁期间SBP测量的时间加权平均值。通过精确匹配与距离匹配法，以10名BHS参与者匹配1名CARDIA参与者，匹配变量为两队列测量年龄重叠的早期SBP与认知关系中的假设中介变量与混杂因素，包括社会人口学特征及中年期血压、认知与血管风险因素测量值。采用混杂因素调整的随机截距模型评估早期SBP与认知结局的关联。结果：早期SBP与中晚年认知原始分无显著关联：执行功能[β（95% CI）= 0.285（-0.867, 1.437）]、记忆力[-0.019（-0.308, 0.269）]、处理速度[-0.188（-1.611, 1.236）]。采用Z分数认知结局时亦未观察到关联。结论：未发现早期SBP与中晚年认知存在关联的证据，但该结论的解读依赖于合成队列方法中关于队列可比性与链接有效性的强假设。尽管结果缺乏精确性，我们观察到预期方向的潜在信号（即收缩压升高导致所有认知领域功能下降），这或可成为未来研究的重点。
    队列概况：用于早期妊娠中期高风险孕妇模型外部验证的风险预测模型评估队列（CORE） Jain, R. s. 2026-07-06 PDF 目的：风险预测模型评估队列（CORE）旨在通过外部验证识别孕早期高危妊娠的预测模型。这类模型通常基于小规模单源数据集开发，且很少在其他人群中进行测试。最新证据显示，仅约6-10%的模型曾接受外部验证，这引发了对模型在新异人群中可靠性的担忧。在围产期领域，CORE通过提供印度孕中期队列数据填补这一空白，该队列包含标准化影像与结局数据，可用于验证现有风险预测模型。参与者：CORE纳入964名年龄超过18岁的孕妇，于2021年8月至2023年3月期间在新德里哈姆达德医学科学研究所（HIMSR）入组。孕妇在孕20周前完成招募，并在孕18-22周接受超声检查及分娩时随访。所有时间点均采集结构化社会人口学、临床及产科数据，同时获取孕18-20周超声图像用于测量胎儿生物测量参数和宫颈长度。当前发现：孕妇中位年龄27.6岁；51%体重指数（BMI）正常，30%超重。初产妇（480例，50%）与经产妇（484例，50%）比例相近。经产妇中既往早产史患病率约41%。750名参与者获得结局数据（23例流产、3例死产、724例单胎活产）。在724例活产中，80/724（11%）为早产，190/716（26.5%）为小于胎龄儿（SGA），40/716（5.6%）为大于胎龄儿（LGA）（依据INTERGROWTH-21st标准），260/716（35.6%）新生儿被归类为小型脆弱新生儿（SVN）。未来计划：CORE将用于外部验证，并与类似队列联合分析，以改进印度及相似环境下孕妇的风险预测模型。我们诚邀合作者使用该资源；临床与影像数据在合理请求下可通过受控访问模式获取。研究优势与局限性：前瞻性队列，数据采集与超声方案与GARBH-Ini和AMANHI队列标准化，支持同类合并及跨队列分析。双层超声质量保证流程保留原始（无标注）及卡尺标注图像，支持基于图像的预测模型验证。样本量基于预测模型外部验证的精度指导原则计算，为评估主要结局的模型区分度提供充分统计效力。单中心医院招募且地理覆盖范围有限，限制直接推广性，需与同类队列合并分析才能发挥最大价值。孕20周至分娩期间无连续随访，限制了对时间变化及远期结局的评估。
    自新冠疫情以来，英格兰内窥镜和影像检查延迟的地理差异：基于月度诊断数据的分析 White, B. 2026-07-06 PDF 目的：新冠疫情期间，英格兰诊断检测延误问题加剧，但目前尚不清楚等待时间恢复速度如何，以及恢复速度是否存在地区差异。本研究旨在：（1）明确影像和内镜检查的延误情况自疫情以来及过去一年是否有所改善；（2）量化各地区（次级综合护理委员会）之间延误的差异（排除随机因素），并评估过去一年差异是否扩大或缩小；（3）判断当前各地区延误趋势是否存在差异。方法：我们分析了英格兰国家医疗服务体系（NHS England）2019年1月至2026年4月期间103个地区的公开月度数据，涵盖八项诊断检测（四项影像、四项内镜）中等待6周及以上的患者比例。描述时间趋势，并通过混合效应二项逻辑回归（含地区随机截距和斜率）比较2026年4月与2025年4月等待6周以上比例的变化。利用模型随机效应方差量化超出随机因素的地理差异及其年度变化，并将结果转换为第25至第75百分位值，以呈现中间50%地区的分布。通过模型条件估计分析各地区特定趋势。结果：尽管疫情后影像和内镜检测等待6周及以上的患者比例部分下降，但过去一年这一改善似乎停滞，甚至胃肠道内镜和磁共振成像（MRI）的延误有所增加。2025年4月至2026年4月，胃镜（22%升至30%；p < 0.001）、结肠镜（25%升至30%；p = 0.002）、可屈性乙状结肠镜（27%升至33%；p < 0.001）和MRI（15%升至25%；p < 0.001）的延误显著增加，其他检测无显著改善。地区间存在超出随机因素的显著地理差异，中间50%地区（即第25至第75百分位）的差异达3至5倍。过去一年地理差异幅度持续存在，且MRI的差异显著扩大。自2025年4月以来，各地区变化差异显著，部分地区延误减少超过50%，而其他地区则翻倍。结论：英格兰诊断检测延误尚未恢复至疫情前水平，关键内镜服务的近期改善已停滞或逆转。地区间存在显著且持续的地理差异，且地方趋势分化。政策应对应优先考虑减少总体等待时间，并解决诊断能力、需求及服务组织方面导致地方波动的根本驱动因素。
    睡眠质量差与加速衰老之间存在显著相关性，但因果关系的证据尚不明确。 Whitman, E. T. 2026-07-06 PDF 睡眠质量随年龄增长而下降，并与疾病风险和死亡率相关。睡眠不佳导致衰老加速的可能性，促使人们关注通过改善睡眠来延缓衰老和预防疾病。然而，关于睡眠不佳与加速衰老之间联系的现有证据尚不明确。本研究利用五个独立成人数据集（总样本量超过64,000人），检验了睡眠不佳与加速衰老之间的相关性和因果关系。我们发现强有力的证据表明，睡眠不佳与快速衰老之间存在相关性，这种相关性在青年、中年和晚年阶段以及源自不同组织和模态的衰老生物标志物中均保持一致。我们还发现，这种相关性不受慢性疾病负担的影响，但受双胞胎间共享遗传和早期环境因素的影响。最后，通过孟德尔随机化分析，我们发现了睡眠不佳对加速衰老存在因果影响的不一致证据。研究结果表明，睡眠不佳与加速衰老之间的相关性非常稳定，但睡眠不佳导致衰老加速的论断并未得到一致支持。

    COVID-19 mRNA疫苗接种降低吉兰-巴雷综合征风险：来自大型纵向队列研究的证据 Li, J. 2026-07-06 PDF 摘要 此前关于COVID-19疫苗接种与吉兰-巴雷综合征（GBS）关联的研究结果不一致，主要受限于数据不足和缺乏同期对照组。为解决这一问题，本研究利用国家COVID队列协作组（N3C）数据开展了一项大规模纵向队列研究。结果显示，COVID-19感染与GBS发生率升高相关，而相对于未暴露（未接种疫苗且未感染）对照组，COVID-19疫苗接种与GBS风险显著降低相关，30天风险降低61%（发病率风险比：IRR = 0.39，P < 0.01），多变量Cox回归分析也证实了类似的风险降低（调整后风险比：aHR = 0.41，P < 0.01）。这种保护性关联仅见于mRNA疫苗接种者（BNT162b2：IRR = 0.38，P < 0.01；mRNA-1273：IRR = 0.24，P < 0.01），而腺病毒载体疫苗接种者未观察到该关联（IRR = 1.38，P > 0.05）。既往COVID-19疫苗接种还可降低感染相关的GBS风险。与GBS风险相关的其他因素包括性别、疫苗剂量以及卒中、神经系统疾病和自身免疫性疾病等既往合并症。总体而言，本项基于N3C的大规模研究提供了COVID-19 mRNA疫苗接种可降低GBS风险的证据，支持mRNA疫苗的安全性特征，并有必要开展进一步机制研究。

    钝性创伤性脑损伤早期死亡预测临床评分系统的比较性能 Abdollahi Sarvi, M. 2026-07-06 PDF 背景：创伤性脑损伤（TBI）的早期风险分层对于入院后关键数小时内的及时分诊、资源分配和临床决策至关重要。本研究旨在比较五种既定临床评分系统——格拉斯哥昏迷量表（GCS）、修正创伤评分（RTS）、机制-格拉斯哥昏迷量表-年龄-动脉压（MGAP）评分、修正早期预警评分（MEWS）和快速急诊医学评分（REMS）——对钝性TBI患者早期死亡预测的性能。方法：这项单中心回顾性观察队列研究评估了2022年3月至2025年3月期间伊朗德黑兰一家三级创伤中心重症监护室收治的444名18至89岁钝性TBI患者。主要结局为早期死亡，定义为入院后24小时内死亡。判别性能通过受试者工作特征曲线下面积（AUC）及基于bootstrap重抽样（1000次迭代）的95%置信区间（CI）进行评估。进行AUC的成对比较，并通过约登指数确定最佳诊断截断值。结果：入院后24小时内，97名患者（21.8%）发生早期死亡，347名患者（78.2%）存活。创伤特异性和神经学评分系统显示出最高的判别能力：RTS准确度最高（AUC=0.676，95% CI：0.617-0.737），紧随其后的是GCS（AUC=0.669，95% CI：0.608-0.727）和MGAP（AUC=0.657，95% CI：0.594-0.724）。一般生理学评分表现较低，MEWS的AUC为0.651（95% CI：0.595-0.707），REMS的判别能力最低（AUC=0.601，95% CI：0.538-0.659）。成对分析证实RTS、GCS和MGAP显著优于REMS，但RTS、GCS和MGAP之间未观察到统计学显著差异。所有评估系统均仅显示出中等水平的整体预测性能（AUC<0.70）。结论：与REMS等一般急诊评分相比，创伤特异性和神经学导向的评分系统（RTS、GCS和MGAP）对钝性TBI的24小时死亡率提供了更优且相当的预后准确性。然而，所有评估模型的绝对预测能力仍属中等。依赖静态入院变量的传统系统无法捕捉继发性脑损伤的动态多因素特性，这凸显了对整合生理时间序列数据或机器学习算法的多维预后工具的迫切需求。
