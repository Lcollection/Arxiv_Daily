<h3>单突触连接将人类皮层中功能相似的区域联系起来。</h3>
<span class="paper-item__meta">bioRxiv / 2026-05-20 / Xu, R.</span>
<p>人类大脑皮层的功能组织已被详细绘制1-5，但连接这些区域的轴突投射仍 largely 未知。这一空白阻碍了我们推断皮层网络中的计算路径，以及理解每个区域为何连接特定目标区域的原则。本研究对接受颅内癫痫监测的患者，采用同步电刺激与功能磁共振成像（es-fMRI）技术（此前已在非人灵长类动物中通过示踪研究验证6），绘制了全脑皮层位点的解剖连接图谱，并将这些图谱与同一受试者的任务态及静息态fMRI数据相结合。Es-fMRI揭示了四项主要发现。第一，连接遵循非对称功能相似性原则：相连位点倾向于具有相似的功能特征，但由于连接稀疏性，大多数功能相似的位点对并未形成连接。第二，尽管连接强度随距离增加而衰减7，但功能特征对连接方差的解释力是距离的三倍，且长程连接具有最高的功能特异性。第三，es-fMRI揭示了已确立功能区域之间的直接单突触连接，包括梭状回面孔区（FFA）2与颞顶联合区（TPJ）3在社会认知中的连接。第四，刺激位点的静息态功能连接（rsFC）与该位点的es-fMRI连接仅呈弱相关。综上，这些结果首次建立了人类皮层解剖连接与功能组织之间的原则性关联，并确立了es-fMRI作为构建示踪级人脑连接组的可扩展方法。</p>

<h3>小鼠逃逸回路中，突触前时间动态灵活设定输入权重。</h3>
<span class="paper-item__meta">bioRxiv / 2026-05-20 / Tan, Y. L.</span>
<p>面临威胁的动物必须整合多重信息流——关于危险、环境和内部状态——并在时间压力下做出逃生决策。在小鼠中，这一计算由背侧导水管周围灰质（dPAG）的谷氨酸能神经元执行，但其汇聚的长程输入如何协同驱动灵活决策尚不明确。本研究发现，每个输入的功能权重主要由其突触前活动的时间统计特性决定，而非通路身份或突触位置。我们首先在自然行为过程中进行多区域单单元记录，并利用广义线性模型估算中脑、下丘脑和皮层输入至dPAG神经元的功能连接。随后结合突触分辨率回路追踪、双光子树突刺激与全细胞胞体及树突记录，以及生物物理建模，识别设定这些权重的机制。我们发现dPAG神经元电致密性紧凑，树突树各处的输入在胞体产生大致均匀的反应。因此，突触前放电动态——神经元内的爆发性及群体同步性——成为输入效能的主要决定因素。这一时间统计框架解释了不同输入区域功能连接的测量差异，并预测当突触前时间结构发生变化时输入权重应动态调整——我们通过展示动机冲突期间皮层输入的快速情境依赖性权重再分配证实了这一点。我们提出，dPAG神经元的亚细胞特化使其能够整合来自分布式来源的信号形成单一决策，且输入权重可在行为时间尺度上灵活调整——这一原则可能适用于其他执行生存决策的脑枢纽。</p>

<h3>人类海马体中theta振荡的视觉空间编码</h3>
<span class="paper-item__meta">bioRxiv / 2026-05-20 / Rostowsky, K.</span>
<p>海马体被认为支持视觉处理与感知，这一观点挑战了长期以来强调导航或陈述性记忆的理论。视觉处理理论的一个关键预测是，海马体应表现出与高级视觉新皮层区域相似的视觉空间编码特性，例如对视觉刺激大小的敏感性以及对侧视野偏好。我们通过颅内脑电图在视网膜拓扑映射任务中测量海马体神经活动，验证了这些特性。海马体在整个任务中表现出特征性的慢波（约2赫兹）和快波（约8赫兹）θ振荡。快波θ对视觉刺激的存在有反应，但对刺激量无反应。相反，慢波θ通常不对刺激存在产生反应，而是随视觉刺激大小呈比例变化，这与更大的感受野一致。慢波θ还表现出对侧偏好，这一效应仅存在于右侧海马体。这些效应均非由微眼跳或同步警觉任务表现所致。这些发现为人类海马体的视野编码提供了电生理证据，支持了强调海马体位于视觉层级顶端的功能理论。此类视觉处理可能与自运动、记忆及其他信号结合，共同支撑海马体θ振荡长期关联的更广泛空间与记忆功能。</p>

<h3>南加州海獭的致死性钩端螺旋体病：病理学发现与问号钩端螺旋体的检测</h3>
<span class="paper-item__meta">bioRxiv / 2026-05-20 / Martinez, M. E.</span>
<p>问号钩端螺旋体波摩那血清型感染在加州海狮中引发周期性暴发，并在海豹中造成零星死亡；然而，同域分布的濒危南方海獭作为沿海健康的重要哨兵物种，其感染频率及相关健康影响尚未明确。鉴于问号钩端螺旋体对其他海洋哺乳动物的广泛影响，本研究旨在筛选特定南方海獭个体进行感染检测，确定钩端螺旋体病是否导致其死亡，并描述钩端螺旋体相关病变。通过回顾性研究（2005-2025年）筛选出19例候选病例进行个案分析，包括钩端螺旋体免疫组化、血清学及聚合酶链式反应检测，重点关注肾脏和肝脏病变。74%（14/19）的疑似病例肾脏PCR检测阳性，其中8例通过DNA序列血清群分型鉴定为问号钩端螺旋体波摩那血清群。14例PCR阳性钩端螺旋体病病例中，7例因肾脏IHC阳性及中重度肾小管间质性肾炎被判定为致死性病例。所有致死病例均检测到抗问号钩端螺旋体波摩那血清型抗体滴度≥1:25,600。其余7例PCR阳性病例因肾脏病变轻微和/或与钩端螺旋体无关且IHC阴性，被归为非致死性钩端螺旋体病。非致死性感染病例的波摩那血清型抗体滴度从阴性到低阳性（1:400）不等。PCR阴性病例的钩端螺旋体抗体滴度均为阴性。致死病例的肾脏大体病变常不明显，或表现为粟粒状白色皮质病灶。组织学病变包括肾小管间质性肾炎、急性肾小管坏死及化脓性肾小管炎（伴肾小管内细菌），病变区域IHC染色显示钩端螺旋体抗原阳性。肝脏大体病变同样不明显，组织学病变罕见，仅1例出现肝细胞解离，2例海獭通过IHC检测到少量钩端螺旋体抗原。多数感染钩端螺旋体的海獭（71%，10/14）在加州降雨量较高的月份搁浅，提示可能存在陆源宿主向海洋的传播途径。鉴于上述发现，且同域分布的加州海狮及邻近流域陆生哺乳动物已确认感染问号钩端螺旋体波摩那血清型，针对海洋与陆生疾病传播动态的专项研究或可为减少南方海獭死亡提供新依据。</p>

<h3>计算机断层扫描在诊断颞下颌关节骨关节炎相对于组织病理学发现的准确性——一项针对41匹马的离体研究</h3>
<span class="paper-item__meta">bioRxiv / 2026-05-20 / Jasinski, T.</span>
<p>计算机断层扫描（CT）用于支持马颞下颌关节（TMJ）疾病的诊断，但其准确性尚不明确。本研究旨在阐明TMJ的CT表现与骨关节炎（OA）组织病理学特征之间的关系。对50具马尸体头部中的82个TMJ进行CT成像和取样。评估CT及组织病理学结果，分析其频率分布、相关性以及CT表现对TMJ OA诊断的贡献。79%的关节存在CT表现，包括反映年龄相关改变的CT解剖变异。CT表现呈现年龄相关分布和相关性，尤其在CT解剖变异类别中。仅50%的关节同时出现CT表现和OA组织病理学特征，证实并非所有CT异常均提示疾病。将所有CT表现纳入基于CT的TMJ OA诊断时，特异性为0.41（95% CI: 0.26-0.58），提示假阳性诊断率较高。排除所有CT解剖变异后，敏感性为0.56（95% CI: 0.40-0.72），表明存在大量假阴性诊断。然而，由于归类为CT解剖变异的软骨下骨囊肿与77%的组织病理学结果呈正相关，将其纳入CT诊断后，敏感性提升至0.79（95% CI: 0.62-0.89），特异性提升至0.92（95% CI: 0.80-0.98）。</p>

<h3>城市绿地接触可降低主观压力与生理唤醒水平。</h3>
<span class="paper-item__meta">bioRxiv / 2026-05-20 / Korkmaz, D.</span>
<p>压力是精神障碍的主要风险因素，而城市生活是关键的环境诱因。接触自然环境可能有助于压力恢复和心理健康，但在现实城市移动过程中，绿色空间与灰色空间对生理唤醒和主观压力的影响差异尚不明确。这项预先注册的研究（https://doi.org/10.17605/OSF.IO/HF4RW）采用基于地理位置的可穿戴评估方法，考察了城市绿色与灰色环境转换过程中的心理生理唤醒和主观压力。36名健康城市居民在德国科隆完成了一条平衡设计的环形步行路线，全程记录GPS、心血管和皮肤电活动数据，同时通过生态瞬时评估法采集主观压力、情绪和体力消耗数据。与灰色空间相比，绿色空间与更低的主观压力和更高的情感幸福感相关，心脏指标显示绿色空间暴露期间自主神经唤醒降低。环境转换前后的自主神经变化持续超出即时转换窗口，表明绿色空间暴露的生理益处可延伸至后续的灰色环境。这些发现凸显了城市绿色基础设施在预防压力相关精神健康问题方面的公共卫生潜力。</p>

<h3>对社交互动的猴子进行同步脑记录时，对社会变量的动态追踪</h3>
<span class="paper-item__meta">bioRxiv / 2026-05-20 / Simon, S.</span>
<p>灵长类动物具有高度社会性，但理解其神经基础一直颇具挑战，因为此前仅能记录单个动物与社交伙伴互动时的大脑活动。本研究在自然无约束环境中，同步记录了两只猕猴社交互动时的无线神经活动。每只猕猴的神经活动编码了关键社会变量，如理毛状态、伙伴身份及联合运动。令人惊讶的是，每只猕猴高级视觉皮层的神经元持续追踪社会恩惠信号（即给予与接受理毛的净时长），为理毛互惠行为提供了神经基础。在每只猕猴中，其伙伴的关节运动与神经活动能更准确地预测其自身神经活动，尤其在接受理毛时比给予理毛时更为显著。因此，社交互动的主导者是理毛接受方而非给予方。综上，本研究在生态有效环境中揭示了灵长类社会互动丰富而动态的神经基础。</p>

<h3>YTHDF蛋白塑造了阿尔茨海默病的大脑基因特征</h3>
<span class="paper-item__meta">bioRxiv / 2026-05-20 / Tasaki, S.</span>
<p>阿尔茨海默病（AD）大脑的基因特征反映了遗传、表观遗传、表观转录组和转录后调控复杂相互作用的结果，但塑造这些特征的主导因素仍不清楚。为识别影响AD大脑特征的最关键因素，我们通过可解释的机器学习框架，将细胞与分子特征同差异基因表达相结合。结果表明，YTHDF蛋白（N6-甲基腺苷RNA修饰（m6A）的经典读取蛋白）是AD大脑特征最具影响力的预测因子。我们进一步发现，含YTHDF的蛋白模块在人类AD大脑中表达下调，而在iPSC来源的2D和3D神经元模型中敲低或药物抑制YTHDF蛋白，可重现AD相关的转录特征。此外，eCLIP-seq分析揭示YTHDF蛋白通过m6A依赖和非依赖两种途径影响AD特征。这些结果凸显了YTHDF蛋白在塑造AD大脑基因特征中的核心作用。</p>

<h3>人上皮细胞共培养系统揭示了梅毒螺旋体不同的宿主细胞互作行为</h3>
<span class="paper-item__meta">bioRxiv / 2026-05-20 / Capuccini, K.</span>
<p>梅毒螺旋体苍白亚种（梅毒病原体）因体外培养长期受限而难以研究。尽管近年兔上皮细胞共培养系统（Sf1Ep）取得重大进展，但缺乏基于人类细胞的模型限制了临床相关性与宿主-病原体互作机制解析。本研究旨在建立可支持梅毒螺旋体体外生长的人类上皮细胞系。我们选取六种不同组织来源的人类上皮或上皮样细胞系，在标准梅毒螺旋体培养基的微需氧条件下进行评价。其中CAL-39（外阴）和HepG2（肝脏）细胞系在支持梅毒螺旋体存活、复制、特征性生长行为及长期传代方面达到与Sf1Ep细胞相当的水平。通过延长培养周期，定量分析了梅毒螺旋体的生长动力学、附着动态及运动能力。利用活细胞成像技术，我们首次定义了两种梅毒螺旋体宿主细胞互作行为——表面关联爬行与稳定单极附着。这两种行为在所有测试宿主细胞系中均被观察到，且仅能在支持持续生长的细胞系中长期存在。综上，本研究建立了临床相关的人类上皮细胞共培养模型，揭示了宿主细胞依赖性生长与运动的新机制，为梅毒致病机制研究与疫苗靶点发现提供了技术平台。</p>

<h3>干燥唾液微滴中渗透压变化速率驱动替代呼吸道细菌失活</h3>
<span class="paper-item__meta">bioRxiv / 2026-05-20 / Medina, T.</span>
<p>呼吸道病原体的空气传播取决于其在干燥的呼吸道飞沫中保持活性的能力，然而液滴蒸发过程中细菌失活的物理化学驱动因素仍缺乏定量研究。本研究将受控液滴实验与物理化学建模相结合，探究渗透压动态变化对细菌存活的影响。以大肠杆菌和表皮葡萄球菌分别作为革兰氏阴性菌和革兰氏阳性菌的代表，我们在多种相对湿度条件下测量人工唾液液滴干燥过程中的活性损失，并利用呼吸道气溶胶模型重建时间分辨的渗透压变化。两种细菌在液滴保持液态时均维持稳定，但在发生风化后——此时溶质浓度快速变化导致渗透压急剧升高——活性显著丧失。失活程度与风化过程中渗透压变化速率呈对数线性关系：大肠杆菌的衰亡速度快于表皮葡萄球菌，且在人工唾液中建立的关联性可预测独立磷酸盐缓冲液实验中的存活情况。更快速的湿度下降会导致更剧烈的渗透压冲击和更严重的失活。这些结果证实，风化过程中渗透压变化速率可作为定量且不依赖介质的预测指标，用于评估干燥呼吸道飞沫中细菌的存活能力。</p>

<h3>全球心血管代谢风险结构的变异：基于WHO STEPS数据的146,000名参与者48国比较贝叶斯网络分析</h3>
<span class="paper-item__meta">medRxiv / 2026-05-20 / Babagoli, M. A.</span>
<p>背景 个体层面的心脏代谢性慢性疾病（CMBCD）源于社会人口学、行为学和代谢因素等多层级网络间的复杂相互作用。尽管存在一组共识性风险因素驱动CMBCD，但人群背景会影响风险因素的效应及其相互作用。为深入理解这一现象，我们采用比较建模方法，研究了不同人群中心脏代谢变量的多层级网络结构。方法与结果 利用来自世界卫生组织&quot;非传染性疾病风险因素阶梯式监测调查&quot;中48个国家的全国代表性横截面数据，我们构建了包含社会人口学、行为学及心脏代谢变量（肥胖、糖尿病、高血压、高脂血症和心血管疾病）的国家特异性贝叶斯网络。通过计算网络对间的结构汉明距离，比较了不同区域和收入水平国家间的网络结构差异。随后利用学习到的网络评估个体风险因素对心脏代谢结局的影响及其交互作用。国家特异性贝叶斯网络在直接或间接关联心脏代谢结局的风险因素方面存在差异。不同区域间的网络结构存在显著差异（p=0.023），但不同收入水平间无显著差异（p=0.91）。这些结果在替代学习算法、网络比较指标和数据插补方法下均保持稳健。高龄（60岁以上 vs 30-44岁）与欧洲及中亚地区肥胖概率增幅（+80%）高于其他区域相关。高等教育与南亚地区肥胖（+53%）、糖尿病（+18%）和高血压（+2%）概率增加相关，但在中东和北非地区则与肥胖（-10%）、糖尿病（-32%）、高血压（-16%）和高脂血症（-25%）概率降低相关。年龄与性别在预测肥胖时的交互作用在欧洲及中亚地区国家中占比最高。尽管该数据集提供了多国标准化数据以定义心脏代谢风险因素及驱动因素，但部分健康结局数据有限，且各区域数据可用性不均衡。结论 这些结果揭示了多层级心脏代谢风险结构的具体区域模式，强调需制定区域性公共卫生策略，而非直接套用通用共识性循证模型。未来研究应探索区域间心脏代谢风险因素、驱动因素及疾病相互关联差异的结构性驱动因素。</p>

<h3>一项基于大型电子健康记录数据库的研究：社会和心血管风险因素作为轻度认知障碍向痴呆进展的预测因子</h3>
<span class="paper-item__meta">medRxiv / 2026-05-20 / Miramontes, S.</span>
<p>背景与目的：从轻度认知障碍（MCI）进展为阿尔茨海默病及相关痴呆（AD/ADRD）的个体差异显著，但导致这种异质性的机制尚不明确。识别影响这一转变的临床和社会决定因素有助于实现早期干预。尽管心血管和社会风险因素已被证实是痴呆发病的诱因，但其在MCI向痴呆进展中的作用可能有所不同。目前鲜有研究利用真实世界临床数据评估这些MCI进展的潜在决定因素。方法：基于UCSF Health（2010-2024年）新发MCI患者的电子健康记录（EHR），我们评估了心血管因素（血压[BP]、体重指数[BMI]和2型糖尿病）与社会因素（婚姻状况、语言偏好、种族/民族和社区劣势）对MCI向AD/ADRD进展速率的影响。采用协变量校正的Cox比例风险模型估算新发AD/ADRD的风险比，并分析性别交互作用。结果：在6,529名患者中，收缩压升高与AD/ADRD发病相关（每10 mmHg风险比[HR]：1.09，95%置信区间[CI]：1.05-1.14）。BMI与发病风险呈负相关，男性（HR：0.94；95% CI：0.92-0.97）和女性（HR：0.98；95% CI：0.96-0.99）均如此。与已婚者相比，丧偶患者的进展风险更高（HR：1.15；95% CI：1.00-1.32）。西班牙语使用者（HR：1.38；95% CI：1.04-1.81）、中文使用者（HR：1.19；95% CI：1.00-1.42）及其他非英语使用者（HR：1.24；95% CI：1.03-1.51）的进展风险均高于英语使用者。拉丁裔（HR：1.22；95% CI：1.01-1.48）和亚裔患者（HR：1.14；95% CI：1.00-1.30；p=0.04）的进展风险也高于白人患者。社区劣势与疾病进展无显著关联。讨论：心血管和社会因素独立影响痴呆进展，且存在部分性别特异性模式。整合临床与社会指标凸显了EHR数据在早期识别高风险患者、支持公平痴呆预防中的潜力。</p>

<h3>使用经颅磁刺激运动皮层对特发性震颤进行相位靶向调节</h3>
<span class="paper-item__meta">medRxiv / 2026-05-20 / Mancini, V.</span>
<p>神经振荡为协调分布式脑网络内外的信息交流提供了时间框架。在特发性震颤（ET）中，小脑-丘脑-皮质环路内的病理性同步化产生节律性活动，表现为不自主的动作性震颤。尽管深部脑刺激能有效抑制震颤，但其侵入性和成本凸显了对能够选择性调节病理性振荡的非侵入性干预手段的需求。经颅磁刺激（TMS）提供了一种非侵入性激活皮质环路的方法，但传统刺激方案独立于持续的神经动力学进行。这种开环方法忽略了震颤相关活动的时间结构，可能在振荡的放大期和抑制期均施加刺激。为解决此问题，我们比较了两种相位靶向TMS范式：首脉冲锁相TMS（First-pulse-TMS），仅刺激序列的初始脉冲与震颤相位对齐；以及逐周期锁相TMS（Continuous-TMS），每个脉冲均基于实时震颤相位连续触发。十例ET患者通过加速度计记录外周震颤信号，并利用Oscilltrack算法实时估算震颤相位。在震颤周期的九个离散相位区间内施加64串TMS脉冲，每个相位区间重复约七次。Continuous-TMS在连续周期中保持精确锁相（平均锁相值约0.9），而First-pulse-TMS随时间推移出现渐进性漂移且相位一致性低（平均锁相值&lt;0.2）。Continuous-TMS的刺激相位圆浓度显著高于First-pulse-TMS（Mann-Whitney U检验，p&lt;0.001），表明两种方案的整体锁相精度存在显著差异。关键的是，与First-pulse-TMS不同，Continuous-TMS诱导了双向、相位依赖的震颤幅度调节。圆-线性模型显示刺激相位与震颤幅度变化呈正弦关系，震颤放大和抑制分别出现在周期的相反相位。研究控制了基线震颤幅度和试验次数等协变量。部分患者的震颤抑制效果持续至刺激结束后，提示锁相TMS可能成为潜在的治疗工具。通过实现震颤周期可靠、相位特异性的刺激，Continuous-TMS能够识别产生最大震颤抑制效果的个体化相位，为开发个性化、相位特异性的神经调控策略提供支持。这项原理验证研究表明，时间精确的闭环TMS能实时与病理性振荡相互作用，为探究振荡对运动症状的贡献提供了机制框架，并为ET及其他振荡性疾病提供了可扩展的治疗方案。</p>
