<h3>人类肠道微生物组中表达孤儿基因、假孤儿基因和保守基因的鉴定与分类</h3>
<span class="paper-item__meta">bioRxiv / 2026-07-22 / Chen, C.</span>
<p>孤儿基因——在物种外部缺乏可检测同源物的基因——广泛存在于微生物基因组中，被认为有助于微生物的适应与分子创新。然而，并非所有预测的孤儿基因都代表具有功能的新型编码序列。假阳性孤儿基因（又称虚假孤儿基因）可能源于基因预测错误。我们推断，缺乏可检测表达的孤儿基因更可能是虚假的。为验证这一假设，我们结合人类肠道微生物组的大规模宏转录组谱分析及机器学习方法，区分有表达的孤儿基因与虚假孤儿基因，并将其与多物种共有的保守基因进行比较。利用近5000个宏转录组文库，我们鉴定出约21.8万个具有表达证据支持的孤儿基因，而约33万个预测的孤儿基因缺乏可检测表达，被归类为虚假基因。我们为每个基因提取了154个序列、结构及进化属性特征，并在考虑基因组代表性后训练了XGBoost分类器。模型在区分有表达孤儿基因与虚假孤儿基因时，受试者工作特征曲线下面积（AUC）达0.82；在区分有表达孤儿基因与保守基因时，AUC达0.93。基于SHAP的解读揭示了清晰的生物学信号：特别地，有表达孤儿基因比虚假孤儿基因存在于更多基因组中，且其长度短于保守基因。本研究改进了孤儿基因的发现方法，并表明有表达孤儿基因在序列组成、结构约束及进化信号方面与保守基因及虚假孤儿基因存在系统性差异。</p>

<h3>拟南芥基因组内容快速分化的遗传控制</h3>
<span class="paper-item__meta">bioRxiv / 2026-07-22 / Fiscus, C. J.</span>
<p>真核生物的基因组进化主要由重复序列的动态变化驱动，这些重复序列在拷贝数和序列组成上差异显著。重复序列的进化速率在物种间和物种内部均存在差异，并可能受到遗传和环境因素的共同调控。为揭示影响基因组内容进化速率的因素，我们采用基于K-mer的新方法分析了1142个拟南芥重测序基因组，通过表征基因组内容变异并识别重复序列丰度差异背后的超变区域。随后，我们将重复序列丰度作为数量性状，对超过400个重复家族进行全基因组关联分析，以确定拷贝数变异的遗传基础。通过元全基因组关联分析整合这些结果，我们发现了顺式作用变异和50多个调控全基因组重复序列丰度的反式作用位点。顺式作用变异主要集中于着丝粒周围和着丝粒区域，而反式作用位点则富集于参与DNA复制、DNA修复和DNA甲基化调控的候选基因。最后，我们获得证据表明纯化选择会抑制加速基因组内容分化的突变，倾向于选择限制重复序列扩张的等位基因。这些发现为拟南芥基因组进化的遗传架构和进化驱动力提供了新见解，并为研究其他植物物种中的类似过程建立了分析框架。</p>

<h3>人类线粒体接触位点和嵴组织系统（MICOS）亚复合物的整合结构测定</h3>
<span class="paper-item__meta">bioRxiv / 2026-07-22 / Jindal, M.</span>
<p>线粒体接触位点与嵴组织系统（MICOS）复合体是存在于嵴连接处的线粒体内膜（IMM）组装体，负责调控嵴的形成与重塑，但其结构尚不明确。我们采用贝叶斯整合结构测定方法，结合AlphaFold预测、交联质谱数据、生化实验、电子断层扫描、同源建模及序列比对，对包含Mic60、Mic19、Mic10和Mic13的MICOS复合体结构进行了表征。整合结构揭示了Mic10N,C、Mic60LBS1,LBS2、mitofilin与Mic13central,C之间新的相互界面，并通过实验验证。多个可能致病的错义突变也定位于这些新界面，凸显其重要性。结果表明Mic13可能通过结合IMM近端区域的Mic10和膜间隙中的Mic60促进MICOS组装。综上，我们的整合方法为MICOS复合体的结构与组装提供了新见解。</p>

<span class="paper-item__meta">bioRxiv / 2026-07-22 / HASSANI, I.</span>
<p>庞贝病（糖原贮积症Ⅱ型）是一种常染色体隐性遗传的溶酶体贮积症，其特征为溶酶体内糖原进行性蓄积，导致溶酶体增大、自噬体堆积及自噬流障碍。在病理生理特征中，线粒体异常长期被视为溶酶体功能障碍的继发性后果。电镜观察通常显示其存在副晶格样包涵体、嵴丢失、线粒体肿胀及糖原填充结构。然而，骨骼肌中线粒体与溶酶体的空间组织及相互作用，以及这些改变随肌肉代谢谱的演变规律仍不明确。本研究提出一种新方法，将超分辨率成像与基于深度学习的图像分析流程相结合，定量评估庞贝病主要小鼠模型骨骼肌中线粒体与溶酶体的重塑及其相互作用。根据肌肉类型、纤维类型及线粒体亚细胞定位，在疾病两个特定阶段对细胞器进行分析。结果表明，线粒体网络整体结构在症状前期（1月龄）即已受损，而该阶段线粒体密度变化较为局限，随疾病进展（4月龄）逐渐扩散。重要的是，这些病理生理改变高度依赖于肌肉类型、纤维类型及亚细胞定位。伴随溶酶体体积快速广泛增大及后期溶酶体聚集更紧密，我们观察到线粒体-溶酶体相互作用呈渐进性、区域特异性增强，以肌原纤维间区域最为显著。本研究证实，这种原创性成像方法为定量分析骨骼肌纤维内细胞器相互作用提供了相关且强大的框架，为探索疾病进展的亚细胞变化开辟了新途径。因此，该方法可作为监测病理生理过程及评估治疗干预效果的有力工具。关键词：庞贝病，线粒体，溶酶体，超分辨率，图像分析</p>

<h3>癫痫相关基因的表达模式作为PTE和SE模型中共同的病理特征，以及通过计算机模拟方法探索可能的治疗策略</h3>
<span class="paper-item__meta">bioRxiv / 2026-07-22 / Tyagi, J.</span>
<p>从初始脑损伤到慢性癫痫的转化仍是临床神经科学中的关键挑战，常涉及难以通过单基因分析捕捉的复杂分子变化。本研究探讨了两种不同病因模型——铁诱导（FeCl3）创伤后癫痫（PTE）和锂-匹罗卡品诱导（LiCl-Pilo）癫痫持续状态（SE）——中基因组不稳定性是否存在汇聚通路。通过整合电生理（EEG）、γ振荡频谱分析、多单元活动（MUA）记录以及行为学评估（Morris水迷宫MWM和旷场实验OFT），我们验证并建立了这两种模型的慢性过度兴奋性及相关认知-情感共病。这进一步为研究由GABRA2、KCNN2、KCNAB1、GRIK1、BCL11A、BRD7、STX1B和PNPO组成的靶向8基因组合提供了功能背景。RT-PCR分析显示，这8个基因中有5个出现系统性失调，反映了人类表型中抑制性信号传导、膜稳定性和神经递质代谢的同时崩溃。尽管这些基因的临床作用已有充分记载，但我们的研究表明，这些大鼠模型为复杂多基因研究和基因水平治疗探索提供了高保真平台。此外，计算机对接发现非瑟酮（Fisetin）是失调蛋白的高亲和力配体，为稳定癫痫基因组提供了有前景的多靶点策略。</p>

<h3>土壤食物网中的顶级捕食者提高了碳循环效率。</h3>
<span class="paper-item__meta">bioRxiv / 2026-07-22 / Lejoly, J. D. M.</span>
<p>土壤微生物被认为是凋落物分解和土壤碳形成的核心。微生物的活动和丰度受到微生物捕食者的控制，而这些捕食者本身又被顶级捕食者捕食。然而，顶级捕食者在碳循环中的作用很少被研究，尤其是在以食细菌生物为主的食物网中。我们在此测试了由微生物、微生物捕食者（以食细菌线虫为主的线虫群落）和顶级捕食者（以线虫为食的螨类）组成的营养级联如何影响碳循环及相关微生物库和过程。我们发现，我们的模型顶级捕食者减少了食真菌线虫的丰度，并对微生物组组成产生了级联效应，特别是增加了革兰氏阳性菌的生物量，从而促进了细菌能量通道。这些营养级联效应传导至碳循环，使异养呼吸减少10%，同时保持了凋落物分解速率。综合来看，我们的结果表明，顶级捕食者提高了碳循环效率，并强调了复杂营养相互作用（包括营养级联）在决定土壤碳循环中的重要性。</p>

<h3>靶向调控IGFBP5/IGF1、THPO和P38 MAPK信号通路是适用于线粒体呼吸链疾病和骨肉瘤的强效治疗策略。</h3>
<span class="paper-item__meta">bioRxiv / 2026-07-22 / Keith, K.</span>
<p>原发性线粒体疾病（PMD）目前仅有针对400多种离散基因疾病中3种的疾病修饰疗法。我们此前报道过环己酰亚胺（CHX）作为一种全局性胞质翻译抑制剂可挽救PMD临床前模型，但其毒性阻碍了临床开发。为明确CHX治疗PMD获益的特异性介质，本研究在复合物I缺陷及遗传性疾病的成纤维细胞系模型中（以半乳糖培养）开展基于SOMAscan的蛋白质组学分析。与健康对照细胞相比，CHX处理PMD细胞后仅鉴定出两种差异调控蛋白——促血小板生成素（THPO）和胰岛素样生长因子结合蛋白5（IGFBP5），同时伴随ERK/MAPK通路失调。通过siRNA或药理学方法抑制THPO可挽救具有不同PMD基因病因的患者成纤维细胞中应激诱导的活力丧失，并在经典ndufs2-/-秀丽隐杆线虫模型中显著改善线粒体应激线性生长和神经肌肉功能。通过慢病毒或mRNA方法过表达IGFBP5可挽救不同PMD基因病因的细胞活力，而IGF1药理学抑制在PMD突变细胞和线虫模型中均产生类似效果。MAPK药理学抑制可挽救多种复合物I缺陷细胞存活，并缓解SLC25A46-/-线虫的线粒体应激。针对这些葡萄糖信号通路蛋白的联合疗法（联合葡萄糖和N-乙酰半胱氨酸）在复合物I缺陷细胞和线虫模型中展现出更优治疗效果。此外，单用或联用THPO或IGF1药理学抑制剂可显著增强原发性和转移性骨肉瘤细胞死亡。综上，靶向THPO、IGF1或MAPK的小分子及基因调控手段在避免全局翻译抑制的同时，重现了CHX对PMD的显著治疗效益。这些新型PMD疗法可能通过减弱MAPK驱动的自噬并潜在促进非经典葡萄糖摄取来改善细胞能量平衡。总体而言，这些葡萄糖信号细胞通路靶点为PMD患者提供了广阔的治疗前景，值得进一步开展临床研究。</p>

<h3>HPRC2：一个近乎完整覆盖常见遗传变异的人类泛基因组参考</h3>
<span class="paper-item__meta">bioRxiv / 2026-07-22 / Lucas, J. K.</span>
<p>泛基因组参考通过整合群体中的变异，克服了任何单一参考基因组固有的局限性。我们发布了人类泛基因组参考联盟（HPRC）第二版（HPRC2），这是一个公开可用的第二代泛基因组，其基因组数量较HPRC第一版（HPRC1）扩展约五倍，并在基因组完整性、连续性和准确性方面实现了可量化的提升。通过采用优先覆盖常见变异的算法原则性筛选样本，HPRC2贡献了460个单倍型，共同捕获了&quot;全民健康研究项目&quot;v8队列中超过99%的常见变异。结合高覆盖度的长读长和超长读长测序技术，配合现代组装与抛光工具，我们生成了数千条端粒到端粒（T2T）染色体，相较于HPRC1，每个单倍型的结构不可靠区域数量及单碱基错误率均减半。我们为这些组装结果补充了全基因组多重比对和基因注释，并推导出用于处理非参考变异的正式泛基因组坐标系统，证明单个人类基因组中存在超过十万个无法通过现有参考基因组简洁描述的变异。同时，我们首次展示了同等规模下基于长读长的泛转录组和泛表观组数据，提供了覆盖每个基因组的连续局部祖先推断，并概述了利用该泛基因组资源改进基因组分析的一系列新工具与应用。</p>

<span class="paper-item__meta">bioRxiv / 2026-07-22 / Fazal, A.</span>
<p>萜类化合物是存在于生命三域中普遍且结构多样的天然产物。其中最常见且研究最深入的萜类化合物之一是挥发性、具有雨后泥土气息的土臭素，人类对低至万亿分之几浓度的这种土壤气味物质都能感知。对细菌土臭素合酶（GeoA）编码基因簇的生物信息学分析显示，许多基因簇与编码2B家族包囊体壳蛋白的基因共定位。本研究聚焦于模式黏细菌黄色黏球菌中编码包囊体的土臭素基因簇，发现两种壳蛋白形成双组分2B家族包囊体，并以GeoA作为内化货物。结构分析表明，该包囊体壳含有特征性的外部环磷酸腺苷（cAMP）结合折叠结构域（CBDs），且混合壳可呈现此前未观察到的闭合双孔构象，这可能对货物功能调控具有重要作用。我们证实GeoA货物装载由重复的短货物装载肽（CLPs）介导，且GeoA的包囊化有利于提升酶活性与稳定性。本研究拓展了已知参与特殊代谢产物生物合成的2B家族包囊体清单，并为通过孔道状态调控货物蛋白的潜在机制提供了见解。</p>

<h3>AI驱动的靶向TSLPR异二聚体界面的纳米结合物设计以抑制2型炎症信号传导</h3>
<span class="paper-item__meta">bioRxiv / 2026-07-22 / Ahn, W.-C.</span>
<p>异常胸腺基质淋巴细胞生成素（TSLP）信号传导是2型炎症性疾病的核心驱动因素，然而目前唯一获批的TSLP靶向疗法是分子量为150 kDa的单克隆抗体，其庞大体积限制了组织渗透性并无法实现吸入给药。在此，我们报告了一种AI驱动框架，用于设计超紧凑的从头纳米结合体，通过空间阻断TSLPR-IL-7R异二聚体的组装来抑制TSLP信号传导。我们比较了两种结构引导策略：纯从头螺旋束生成，以及将天然结合基序界面模拟移植到设计支架上。尽管两种方法均产生纳摩尔级结合体，但只有模拟天然细胞因子几何结构的正构位点才能阻断受体异二聚化，表明功能性拮抗作用由精确的表位几何结构而非单纯亲和力决定。经过文库成熟化后，先导纳米结合体TRB5.1是一种超稳定单体（Tm ~97.4°C），具有个位数纳摩尔级亲和力（KD = 9.7 nM），并对相关γ链白细胞介素受体具有严格选择性。TRB5.1在多种细胞模型中抑制TSLP诱导的JAK1和STAT5磷酸化，并驱动全转录组水平的致病性2型程序逆转。这项工作提供了一种可开发、潜在可吸入的非抗体先导分子，以及拮抗异二聚体细胞因子受体的可扩展蓝图。</p>

<h3>足球运动员的肌肉稳定性缺陷与肌肉骨骼问题密切相关：适应性力量比优于传统力量参数——一项含初步随访的横断面研究</h3>
<span class="paper-item__meta">medRxiv / 2026-07-22 / Schaefer, L. V.</span>
<p>鉴于足球运动的高损伤负担以及基于力量的筛查方法已被证实的局限性，亟需探索新方法。适应性力量（AF）——比推/拉力量更接近易受伤动作——提供了一种替代方案。本研究考察了基于AF的肌肉稳定性与足球运动员肌肉骨骼不适之间的关联，并比较了AF衍生参数与传统力量参数的区分能力，同时辅以初步前瞻性随访。方法：对23名男性足球运动员的双侧五组肌肉群（膝伸肌/屈肌；髋屈肌/内收肌/外展肌）测量AF和最大自主等长收缩（MVIC）。通过基线问卷和六个月随访（n=13）评估有无不适的运动员，比较AF参数（最大等长AF、最大AF、AF比值）、MVIC及腘绳肌与股四头肌比值（H:Q）。结果：稳定性缺陷与不适显著相关（OR=54.0，82%侧别一致性）。AF比值能清晰区分有无不适的运动员（d=-1.47），其中髋外展肌效应最强（d=-1.64）。后续出现不适的运动员基线AF比值较低（d=-1.45），稳定性缺陷更多（d=1.67）。MVIC和H:Q比值无区分能力（p&gt;0.430）。结论：肌肉稳定性评估在区分有无不适的运动员方面优于传统力量参数，初步随访数据为预测价值提供了初步支持。功能性不稳定综合征（FIS）概念为非接触性损伤和肌肉骨骼不适提供了机制框架。AF评估在筛查（包括重返赛场决策）方面具有潜力。需进一步研究验证结果、探究预测价值，并评估基于稳定性的个性化干预能否降低损伤发生率。关于本主题的已知信息：传统基于力量的筛查对足球损伤的预测价值有限，荟萃分析中无力量参数达到显著性。AF评估适应性保持能力（肌肉稳定性）——一种比传统力量更接近易受伤动作的神经肌肉功能；其在运动员（尤其是足球运动员）中与肌肉骨骼不适的关联此前未知。本研究新增内容：稳定性缺陷（保持能力受损）与肌肉骨骼不适显著相关（OR=54.0），并得到初步前瞻性支持，而最大推力力量和H:Q比值无区分能力。髋外展肌与全身各部位不适的关联最强——该肌群在传统筛查中很少被纳入。本研究对研究、实践或政策的影响：AF评估可通过识别传统力量方法无法检测的神经肌肉缺陷，补充或完善当前筛查与预防策略，支持向基于稳定性的筛查转变，但需前瞻性验证。</p>

<h3>司美格鲁肽改变了人类胚胎-子宫内膜界面。</h3>
<span class="paper-item__meta">medRxiv / 2026-07-22 / Apostolov, A.</span>
<p>司美格鲁肽（SE）是一种具有降糖和减重效果的胰高血糖素样肽-1受体激动剂（GLP-1RA），其使用量迅速上升，尤其在育龄女性中。临床前研究提示其通过下丘脑-垂体-卵巢轴改善卵巢功能，但对子宫内膜-胚胎界面的影响尚不明确。本研究发现，GLP-1R在人类子宫内膜中动态表达，仅限于上皮细胞，并在月经周期分泌中期显著上调。SE激活细胞内cAMP信号通路，增强上皮细胞代谢，推动其向氧化磷酸化转变。值得注意的是，即使无类固醇激素预处理，最高浓度SE仍能上调多个子宫内膜容受性标志物；而在激素预处理条件下，相同浓度SE会适度降低关键容受性标志物PAEP/糖蛋白的表达。相反，在缺乏可检测GLP-1R的基质细胞中，SE会破坏蜕膜化过程，诱导内质网应激，并抑制G2/M期细胞周期。人类胚胎模型（类囊胚）表达GLP-1R，并在上胚层和滋养外胚层谱系中发生一致的SE介导的转录重塑，涉及代谢和表观遗传调控变化，但谱系比例未改变。值得注意的是，在激素剥夺条件下，所有测试浓度的SE均能恢复类囊胚附着，但当存在生理性激素预处理时，SE未进一步增强附着。综上，这些发现揭示了区室特异性失衡：SE增强上皮和胚胎代谢活性，但削弱基质细胞对胚胎着床的支持，可能因基质功能障碍对着床产生潜在影响。</p>

<p>目标 本研究旨在考察澳大利亚公众对夏令时（DST）的态度及其感知的健康后果。澳大利亚各州在夏令时实施上的差异，为检验接触夏令时相关健康风险信息如何影响这些态度，以及个体差异如何调节这种影响提供了独特背景。方法 在一项预先注册的随机在线实验中，澳大利亚成年人（n=499）被分配接收关于夏令时的中性信息或强调其负面健康后果的夏令时相关健康风险信息。评估了参与者对夏令时的态度、政策偏好、感知健康后果、既有健康意识及信息可信度感知，随后收集了社会人口学特征。使用微型慕尼黑时型问卷评估了时型。结果 接触夏令时相关健康风险信息降低了夏令时政策支持度，并增加了其感知健康后果，有效消除了未接触此类信息时对夏令时微弱多数的支持。这些效应在既有健康意识较高的个体中更为显著。中介分析显示，对夏令时政策整体态度的转变可通过增强对夏令时相关健康后果的认知来解释，且当信息被认为更可信时，这一间接路径被放大。结论 综合来看，我们的研究结果表明，对夏令时的正面看法与其健康后果认知度较低相关，而可信的健康风险信息通过提高对这些风险的认识来降低支持度。这凸显了基于证据的健康风险信息对于指导澳大利亚及国际上夏令时政策制定的重要性，因为夏令时在这些地区仍存在争议。</p>

<p>虚拟临床试验（VCTs）在改善药物开发流程方面具有显著潜力，但其预测可靠性关键取决于设计决策，而这些决策目前仍缺乏深入理解。本研究探讨了模型复杂度如何影响VCT结果，以及先验参数分布的选择和虚拟患者纳入标准如何影响这些结果。以溶瘤病毒疗法治疗小鼠肿瘤为案例，我们比较了三种复杂度递增的数学模型在两种参数先验（均匀分布与正态分布）和两种纳入方法（接受-拒绝法与接受-扰动法）下的相对层级表现。结果表明，最简单的模型产生的可行群体无法充分覆盖可能的轨迹空间，可能遗漏关键的患者间异质性。然而，超过中等模型复杂度后存在收益递减现象——中等模型与复杂模型在不同给药方案下捕捉到的患者反应范围相似。值得注意的是，接受-拒绝法生成的后验参数分布更倾向于接近所选先验分布，可能过度降低治疗反应中的患者间变异性，尤其在高剂量条件下。相比之下，接受-扰动纳入标准产生了更稳健的结果，对先验假设的敏感性更低。这些发现表明，VCT设计应优先选择具备足够生物学细节以捕捉关键机制但避免不必要复杂性的模型，同时配合能避免过度约束可行群体以匹配可能不切实际的先验假设的纳入标准。</p>

<h3>评估大语言模型在结肠镜检查准备辅助中的表现：合成对话的正确性与多样性</h3>
<span class="paper-item__meta">medRxiv / 2026-07-22 / Kaumenova, T.</span>
<p>背景：结直肠癌是美国癌症相关死亡的主要原因之一，结肠镜检查仍是早期发现和预防的金标准。然而，许多检查因肠道准备不充分而推迟，这种可预防的失败通常源于患者难以理解或遵循书面准备说明。此前诸如提醒应用程序和教学视频等干预措施仅能适度提高依从性，主要原因在于它们无法回答患者的具体问题。大型语言模型的最新进展为开发能提供交互式支持的对话助手提供了可能，可帮助患者进行手术准备。目的：本研究评估了由领先大型语言模型生成的模拟AI教练与患者之间关于结肠镜检查准备的合成对话的正确性、危害性和多样性。方法：使用五个领先大型语言模型——OpenAI的o3、GPT-4.1和GPT-5.1，Meta的Llama 3.3 70B，以及Mistral的Large-2411——每个模型生成250段患者-AI教练对话。对话包含3-7个关于饮食、药物及其他准备相关主题的问答对。采用多提示、多问题的方法设计以引发多样化的患者问题，并建立错误分类法评估模型回答问题的能力。包括三名医学专家在内的人类评估员评估生成问题的难度以及回答的正确性、错误类型和潜在危害性。使用LLM作为评判者的自动评估方法补充了人类评估。通过词汇多样性指标（Distinct-1、Distinct-2）和熵评估问题多样性。此外，我们评估了一种安全过滤机制，即自动评估器判断为错误的回答将被替换为指示患者联系医疗保健提供者的推迟消息。使用在对话层面进行的置换检验评估各模型间回答正确性的差异。人类评估员间的评分者一致性使用Gwet的AC1统计量评估。研究于2025年5月至9月进行。结果：自动评估结果与人类判断高度一致：领先模型接近但未达到足够性能。封闭权重模型（GPT-5.1、GPT-4.1和o3）在正确性上优于开放权重模型（Llama、Mistral），其中推理模型（GPT-5.1和o3）表现最佳。这种回合级排名在补充单提示基线中得以保持，但对话级排名有所不同。所有模型均产生有害错误，主要源于准备说明的遗漏或误解。与单提示基线相比，多提示生成策略显著增加了患者问题的多样性。应用自动安全过滤器降低了总体错误率，但未能消除有害回答。结论：尽管大型语言模型在结肠镜检查准备支持方面展现出强大潜力，但尚无模型足够可靠以在面向患者的环境中无监督部署。持续存在的有害错误以及简单过滤机制的有限效果，凸显了改进指令遵循、加强安全机制以及使用真实患者查询进行验证的必要性。</p>

<h3>Boora，一个用于巴西初级保健中超重与肥胖护理的AI辅助数字平台：一项关于感知可用性与可接受性的形成性混合方法评估</h3>
<span class="paper-item__meta">medRxiv / 2026-07-22 / Couto, F. d. F. S.</span>
<p>目的评估Boora（一款面向巴西初级保健中超重与肥胖长期管理的人工智能辅助、人工监督数字平台原型）在用户和卫生专业人员中的感知可用性、可接受性和用户体验（而非临床有效性）。设计采用收敛混合方法形成性评估。感知可用性通过系统可用性量表（SUS）测量并描述性总结；在实操体验后开展半结构化访谈，采用编码本主题分析法（Braun和Clarke）分析；两股数据通过联合展示整合。定性报告遵循定性研究综合报告标准（COREQ）。环境巴西统一卫生系统内帕拉州阿纳宁德瓦初级卫生保健网络（2026年1月至2月）。参与者15名超重或肥胖成人（BMI≥25 kg/m²，经电子健康记录确认），使用患者端应用24小时；8名初级保健专业人员（护士、医生和营养师），使用专业仪表板约20分钟完成预设合成数据任务。主要结局指标SUS评分及涵盖可用性、可接受性、感知有用性、障碍、临床与工作流程适配性的定性主题。结果Boora在两队列中均显示良好感知可用性（用户均值76.5，SD 10.3；专业人员均值77.5，SD 4.6；均高于SUS常模均值68）。每队列各浮现四个主题。用户重视界面易用性和进度可视化，但指出每日记录负担、预期参与度脆弱性及数字素养与可及性障碍。专业人员重视清晰界面及面板管理式主动随访前景，同时提出培训、AI治理、保护性时间及与国家记录互操作性需求。整合分析表明，用户预期的参与度下降正是专业人员认为仪表板可帮助识别的风险，而教育型AI助手对两组均为最薄弱且最模糊的组件。结论Boora被认为可用且可接受，其感知价值集中于人工监督的长期随访，而非自主自我追踪或AI建议。这些发现涉及感知可用性和可接受性，而非临床有效性或持续参与度。实际应用将取决于可及性改进、电子记录整合，以及符合巴西拟议风险导向AI框架和《通用数据保护法》（LGPD）原则的明确AI治理。关键点O_ST_ABS关于本主题的已知信息C_ST_ABSO_LI移动和网络应用可支持体重管理，但缺乏结构化专业支持时用户参与度随时间下降，且多数证据来自高收入国家。C_LIO_LI初级保健中的自主AI建议可能不安全或与临床指南不一致，凸显人工监督和可审计系统的必要性。C_LI 本研究新增内容O_LI在巴西公共卫生系统中，基于短期实操体验，患者和初级保健专业人员均认为配对患者端应用与专业仪表板的人工智能辅助、人工监督平台原型可用且可接受。C_LIO_LI混合方法整合表明，用户预期的参与度下降对应专业人员认为仪表板可帮助识别的风险，将平台感知价值定位于人工监督的长期随访；教育型AI助手是其最薄弱组件。C_LI 本研究对研究、实践或政策的影响O_LI支持将初级保健肥胖工具中的AI设计为可审计的监督支持而非自主建议，符合巴西拟议风险导向AI框架原则。C_LIO_LI参与者指出低识字用户可及性、国家电子健康记录互操作性、保护性时间和培训是潜在采用前提，需开展纵向真实世界评估。C_LI</p>

<h3>驱蚊产品综合评估：功效、安全性及公共卫生影响</h3>
<span class="paper-item__meta">medRxiv / 2026-07-22 / shafoyat, u.</span>
<p>登革热、基孔肯雅热和疟疾等蚊媒疾病仍是南亚地区（尤其是孟加拉国）的重大公共卫生挑战，当地广泛使用驱蚊剂作为主要预防工具。本研究对常用驱蚊产品（包括蚊香、液体电蚊香、气雾剂、避蚊胺类乳膏及天然配方）进行了比较评估，重点分析其功效、保护时长及潜在健康风险。通过实验室控制方法评估功效，包括基于暴露仓的试验和针对埃及伊蚊的臂笼测试（Barnard &amp; Xue, n.d.）。安全性评估采用大鼠模型的体内毒理学分析，涵盖临床观察、血液学与生化指标检测及组织病理学检查。结果显示总体平均有效率为85%，其中避蚊胺类乳膏功效最高（95%），天然驱蚊剂最低（70%）。蚊香保护时长最长（约10小时），但因燃烧排放物导致健康风险最高。电蚊香和气雾剂功效中等且健康风险适中。毒理学发现表明，蚊香暴露会引发显著呼吸应激、肝酶（ALT、AST）升高、白细胞计数增加及明显肺组织损伤。电蚊香和气雾剂暴露仅引起轻微生理变化，而避蚊胺类及天然驱蚊剂全身毒性最小。总体而言，研究揭示了不同驱蚊剂类型在功效与安全性之间的明确权衡。这些发现强调需进行知情产品选择、加强监管控制，并开发更安全、经济且可持续的驱蚊技术，以有效控制流行地区的病媒传播。</p>
